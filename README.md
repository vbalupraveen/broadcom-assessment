# broadcom-assesment

### How to run:
`cd broadcom-assessment-java`
`mvn spring-boot:run`
`cd broadcom-assessment-ui`
`ng serve --open`

### Features
- Show Users
- Search by LastName and/or Age
- Sort By Name or Age

### API
http://localhost:8080/myapp/users?lastName=&age=&currPageIndex=0&pageSize=100&nameSort=null&ageSort=asc
